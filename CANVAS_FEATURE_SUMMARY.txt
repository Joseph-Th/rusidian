KEY DESIGN DECISIONS
================================================================================

1. CANVAS AS VIEW (not new data type)
   ✅ Preserves AI safety invariants
   ✅ No hidden data in JSON blobs
   ✅ All content remains searchable
   ✅ Open, portable format

2. NODES REFERENCE REAL OBJECTS
   ✅ Via ObjectRef (Note, Block, Entity, Media)
   ✅ No data duplication
   ✅ Single source of truth
   ✅ Backlinks work correctly

3. EDGES REPRESENT LINKS
   ✅ When user drags edge: creates real Link
   ✅ Semantic types (Supports, Contradicts, etc.)
   ✅ Visual routing separate from meaning
   ✅ Audit trail preserved

4. LAYOUT AS VIEW PARAMETERS
   ✅ Spatial coordinates stored as JSON
   ✅ No new database tables
   ✅ Versioning through View version number
   ✅ No schema drift
